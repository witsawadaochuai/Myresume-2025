    OctaPoem-AI

    Description: This project is a Flask-based web application designed to assist with Thai poetry or lyric writing by suggesting rhyming words. It leverages the pythainlp library for Thai language processing tasks such as word tokenization, transliteration, and accessing a Thai word corpus. Additionally, it integrates a pre-trained Thai masked language model (wangchanberta) to provide AI-generated rhyming suggestions.

    The application allows users to input up to four lines of text. For each line, it identifies the last word and then offers a variety of rhyming suggestions for the subsequent line. These suggestions are generated from three main sources:

    • • AI Model: Words suggested by the wangchanberta model based on context.
    • • Pronunciation-based Rhymes: Words that rhyme based on their phonetic ending.
    • • Vocabulary-based Rhymes: Words found in a custom vocabulary file that match the target word's sound and form.

    • • Python: The primary programming language for the entire application.

    • PyThaiNLP: A powerful Python library for Thai Natural Language Processing. It's used for:
      • • Word Tokenization: Breaking down sentences into individual words (word_tokenize).
